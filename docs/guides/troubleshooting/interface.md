If a module is not recognized by Grid Editor, the side connector pins may have become misaligned. This guide explains how to inspect and gently realign the pins to restore communication.

1. Identify which side is not connecting
2. Remove the potentiometer/encoder caps (e.g., using a guitar pick to lift them off)
3. Open the faceplate using a screwdriver
4. You will see the pins and a black plastic cover
5. Hold the side interface board and slightly push out the black cover, this will gently bend the pins back into place
6. Test whether it now appears in Grid Editor
