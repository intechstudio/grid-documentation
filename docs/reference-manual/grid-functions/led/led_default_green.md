# `led_default_green`

---

Sets the default green LED component for the module.

```lua
led_default_green(value: integer?): integer
```
<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>value</code> — If provided, sets the green value (0–255)</div>


<div class="doc-tag-line doc-tag-line--return"><span class="doc-tag doc-tag--return">@return</span> <code>green</code> — Current green value</div>
