# `led_animation_rate`

---

Sets the LED animation rate for a specific LED and layer.

```lua
led_animation_rate(
  led_index: integer,
  layer: Layer,
  value: integer
)
```
<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>led_index</code> — Hardware LED index</div>

<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>layer</code> — integer LED layer</div>

<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>value</code> — Animation rate</div>
