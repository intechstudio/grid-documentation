# `led_default_red`

---

Sets the default red LED component for the module.

```lua
led_default_red(value: integer?): integer
```
<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>value</code> — If provided, sets the red value (0–255)</div>


<div class="doc-tag-line doc-tag-line--return"><span class="doc-tag doc-tag--return">@return</span> <code>red</code> — Current red value</div>
