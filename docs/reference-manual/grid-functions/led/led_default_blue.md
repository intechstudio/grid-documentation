# `led_default_blue`

---

Sets the default blue LED component for the module.

```lua
led_default_blue(value: integer?): integer
```
<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>value</code> — If provided, sets the blue value (0–255)</div>


<div class="doc-tag-line doc-tag-line--return"><span class="doc-tag doc-tag--return">@return</span> <code>blue</code> — Current blue value</div>
