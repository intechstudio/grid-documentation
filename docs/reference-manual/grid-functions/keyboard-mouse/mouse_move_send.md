# `mouse_move_send`

---

Sends a mouse movement HID report.

```lua
mouse_move_send(
  axis: integer,
  position: integer
)
```
<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>axis</code> — Movement axis</div>

<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>position</code> — Movement amount</div>
