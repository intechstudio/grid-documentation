# `event_trigger`

---

Triggers an event on a specific element.

```lua
event_trigger(
  element_index: integer,
  event_type: integer
)
```
<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>element_index</code> — Element index</div>

<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>event_type</code> — Event type</div>
