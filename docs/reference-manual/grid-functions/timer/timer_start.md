# `timer_start`

---

Starts a periodic timer for an element.

```lua
timer_start(
  element_index: integer,
  period: integer
)
```
<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>element_index</code> — Element index</div>

<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>period</code> — Timer period in milliseconds</div>
