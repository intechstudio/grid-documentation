# `element_name_set`

---

Sets the name of an element.

```lua
element_name_set(
  element_index: integer,
  name: string
)
```
<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>element_index</code> — Element index</div>

<div class="doc-tag-line"><span class="doc-tag doc-tag--param">@param</span> <code>name</code> — Name to assign</div>
